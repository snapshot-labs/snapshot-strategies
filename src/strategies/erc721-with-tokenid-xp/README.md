# erc721

This strategy return the XP (locked points) of the voters for a specific ERC721 NFT. Pertinant NFTs implement a locked xp reward counter based on timestamp.

Here is an example of parameters:

```json
{
  "address": "0x8fb5a7894ab461a59acdfab8918335768e411414",
  "symbol": "GENESISKEY"
}
```
