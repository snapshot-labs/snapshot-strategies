# hats-protocol-hat-id

Grants voting power based on if voter has a specific hat (IP based).

Here is an example of parameters:

```json
{
  "address": "0x3bc1A0Ad72417f2d411118085256fC53CBdDd137",
  "hatId": "68"
}
```

or 

```json
{
  "address": "0x3bc1A0Ad72417f2d411118085256fC53CBdDd137",
  "hatId": "68.1"
}
```

or

```json
{
  "address": "0x3bc1A0Ad72417f2d411118085256fC53CBdDd137",
  "hatId": "68.1.1"
}
```
