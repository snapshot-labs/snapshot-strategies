# hats-protocol-hat-ids

Grants voting power based on if voter has a set of specified hat (IP based).

Here is an example of parameters:

```json
{
  "address": "0x3bc1A0Ad72417f2d411118085256fC53CBdDd137",
  "hatIds": "[\"68\"]"
}
```

or 

```json
{
  "address": "0x3bc1A0Ad72417f2d411118085256fC53CBdDd137",
  "hatIds": "[\"68\", \"68.1\"]"
}
```

or

```json
{
  "address": "0x3bc1A0Ad72417f2d411118085256fC53CBdDd137",
  "hatIds": "[\"68\", \"68.1\", \"68.1.1\"]"
}
```
