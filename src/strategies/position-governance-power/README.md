# position-governance-power

Here is an example of parameters:

```json
{

  "symbol": "POSI",

  "stakeManagerAddress": "0x0C54B0b7d61De871dB47c3aD3F69FEB0F2C8db0B",
  "nftStakingPoolAddress": "0x6257229FA379AFDBb91732091B5DE32cdB759845"
}
```
